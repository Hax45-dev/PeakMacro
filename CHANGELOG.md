# Changelog

All notable changes to PeakMacro will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release by @Hax25-dev
- Frame-perfect macro recording system
- Macro playback with auto-reset
- Save/Load macros as `.peak` files (JSON format)
- In-level HUD overlay with live REC/PLAY indicator
- Macro Manager UI with tabbed popup (Record | Macros | Settings)
- Keybinds: `P` (open popup), `R` (toggle record), `Y` (toggle playback)
- Pause menu integration with PeakMacro button
- Configurable settings via Geode settings panel
  - Show/hide overlay
  - Overlay opacity
  - Auto-save after recording
  - FPS bypass for playback
- Support for Windows, macOS, and Android
- Macro metadata: name, level name, author, timestamp, fps, frame count
